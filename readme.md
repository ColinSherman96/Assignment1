Linux and GIT command cheat sheet

Linux

ls: list directories

ls -l: list directories with more details

more: view the contents of a file

mkdir: make a directory

mv: move a file (or rename it)

cd: change directory

pwd: print working directory

cp: copy

rm: remove file

rmdir: remove directory

chmod: change mode (r/w/x permissions)

GIT

git init: initialize an existing directory as a git repository

git status: show modified files in working directory

git add [file]: add a file as it looks now to your next commit

git reset [file]: unstage a file. This still retains changes in working directory.

git commit -m "[descriptive message]": commit your staged commit as a new commit snapshot, with description
